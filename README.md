# livewedding

Display live instagram posts for a pre-defined hashtag


### Installing

To get a dev env up in running in seconds, you can use docker
```
docker-compose up
```

Or if you have node and redis on your local machine

```
npm i
npm start
```

## Running the tests

```
npm run test
```
